搜索小专题

DFS部分

DFS是什么

DFS(Depth First Search) 即 深度有限优先搜索 大(D)法(F)师(S) 作为一种图的遍历算法，它的过程可以概括为 一 路 到 底 ,不到黄河（边界条件）不死心。

为什么DFS

  • 能遍历整个图（枚举所有情况）
  • 递归思维
  • 帮你得部分分的靠谱暴力写法 时间复杂度太高
  • 栈的思维 那你为什么不用栈优化
  • DFS得过图灵奖你知道吗

T1 组合的输出

题干

照例挂OJ链接 洛谷P1157 排列与组合是常用的数学方法，其组合就是从 N 个元素中抽出 R 个元素（不分顺序且 R<=N）， 我们可以简单地将 N 个元素理解为自然数 1,2….,N,从中任取 R 个数。 现在要求你用递归的方法输出所有组合。 输入 一行两个自然数 N,R（1<N<21,1<=R<=N）。 输出 所有的组合，每个组合占一行且其中的元素按由小到大的顺序排列，每个元素占三个字符的位置，所有组合也按字典顺序。

思路

组合数和全排列洛谷P1706的思路类似，但是全排列需要考虑例如 _1 3 2_ 和_1 2 3_属于不同排列的情况，因此实际上组合数更符合DFS头铁的逻辑。 组合的本质是不重复地选数，则选到的数即深度，实现深度优先就是在初始选出一个数后，固定该数，在它的基础上选第二个，固定第二个 .etc 边界为选完所有数，即深度达到R，此时输出结果即可return;

BFS部分

BFS是什么

BFS，广度优先搜索,（Breadth First Search）是最简便的图的搜索算法之一。 冰(B)法(F)师(S) 简单的说，BFS是从根节点开始，沿着宽度遍历。如果所有节点均被访问，则算法中止。同样，BFS也是一种比较暴力的求解方式。

如何实现

乍一看BFS很玄乎，因为它用到了队列。 - 1.首先将根节点放入队列中。 - 2.从队列中取出第一个节点，并检验它是否为目标。 - 如果找到目标，则结束搜索并回传结果。 - 否则将它所有尚未检验过的直接子节点加入队列中。 - 3.若队列为空，表示整张图都检查过了——亦即图中没有欲搜索的目标。结束搜索并回传“找不到目标”。 - 4.重复步骤2。 这里我们直接上STL:

为什么BFS

  • 时间复杂度比DFS小 基本上

  • 以后写最短路
  • 顺便学队列

T4 救援

题干

思路

BFS - 1.将起点入队 - 2.弹出一个点 - 若该点为重点，反回步数 - 若不是，向四周扩展，将这些点入队 - 3.若队列非空，回到2 - 4.若队列为空，无法到达 Q.E.D.
