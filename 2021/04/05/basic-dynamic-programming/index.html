动态规划概要

什么是动态规划

动态规划(Dynamic Programming,简称 DP ),是求决策过程最优化的过程。20世纪50年代初，数学家贝尔曼（R.Bellman）(就是发明了Bellman-Ford 最短路那个) 神果然非同凡响 等人在研究多阶段决策过程的优化问题时，提出了著名的最优化原理，从而创立了动态规划。 那么为什么叫动态规划呢？其实这个名字是编出来迷惑你的 这种算法可以解决多段的决策性问题,例如背包问题和Floyd - Warshall 最短路O(n^3) 最丢脸的DP，当然静态问题在人为添加时间后也可以DP解决。 DP的核心逻辑在于每一段做出决策，在更新后不考虑先前的情况。(即DP的无后效性原则) DP的过程：

  • 1.确定每种阶段和方案
  • 2.确定决策的方案
  • 3.状态转移

为什么动态规划

很容易发现，同样是用来求优化决策的贪心算法有一个致命的缺点就是目光短浅，而DP的最优化原理使其能找到全局的最优解而不是部分最优解。 但是DP最难的就在于思路，智商题(误)

01背包问题

题干

有一个包载重为V,有n件价值不同的物品，分别有一个体积和一个重量。 对第i件物品有：价值 V~i~ , 重量W~i~。 求最大拿到的价值是？

方法

容易看出这时候物品不可分割的情况下，贪心明显不太可行，以下三种贪心方法：

  • 1.按最大价格排序

  • 2.按最小体积排序

  • 3.按价值密度（价值和体积的比值）排序

都可以出特殊数据卡掉。这时，就要用无敌的动态规划! 我们令$f{_i,_j}$代表体积为 j 的背包，从前 i 件中选择时得到的最大价值。易证$f{_n,_V}$即为最终答案。 对于每一个物品，只有放和不放两个状态，如果放进去更赚，那就放进去吧！其实对于每一个小部分，DP和贪心没啥区别，但是全局考虑下就不一样了！ 由此可知状态转移方程: $f{_i,_j}=\\max f{_i,_{j - 1}},f{_{i-1},_{j - w{_i}}} + v_i$ 如果放进去第 i 件赚了，$f{_i,_j} = f{_{i-1},_{j-w_i}}+v_i $,即把 第i件放进去,体积增加w~i~ , 价值增加 v~i~。 同理，没放的话，$f_{_i,{j}}$即为放前 i - 1件的情况。 那么为什么贪心不对，DP就是背包问题的正解？ 因为DP具有无后效性原则，即目前已经最优的情况下，由此推导现在的状况不需要再次考虑前边的情况，同时相当于前面所有的情况都给后面的情况排雷，踩坑了，这样才得到全局最优解。一将功成万骨枯

优化

DP的最大问题在于辅助空间太大。只有DP和桶排这么难受 并且我们能发现，这个DP过程中每一步状态转移都不可省略，时间复杂度固定在O(n^2),但是空间复杂度可以优化成O(n),这就要…… 滚动数组 直接使得$f{_i}$表示每一次外层循环得到的最优解，则只需遍历一次即可获得全局最优解。 对于此种情况，有状态转移方程： $f{_j} = \\max f{_j},f{_j - _{w{_i}}+v{_i}}$ 代码

T5 LCS 最长公共子序列

题干

给定两个字符串，求他们的最长公共子序列(以下将最长公共子序列简称为LCS)。 输入 两行，两个字符串。 输出 一个整数，LCS的长度。

思路

求解LCS问题，不能暴力废话。一个长度为n的序列拥有 2^n^个子序列，它的时间复杂度是指数阶。解决LCS问题，还是DP靠谱。 那么，我们就要从最基础的“两个字符串有相等字符”(即有最短长度为1的LCS)这一解得出初始可行解，并且在此基础上进行DP。 令两个字符串A，B有： $A_1$ -> $A_m$ $B_1$ -> $B_n$ 且$LCS(A,B)$ 为Z $Z_1$ ->$Z_k$ 求LCS即求最大k的过程。 可知，对于构造Z有如下法则：

  • 1.如果$A_m$ = $B_n$，则$Z_k$ =$A_m$ = $B_n$，且“$Z_0$，$Z_1$，…，$Z_{k - 1}$”是“$A_0$，$A_1$，…，$A_{m - 1}$”和“ $B_0$， $B_1$，…， $B_{n - 1}$”的一个LCS(对等)

  • 2.如果$A_m$ != $B_n$，则$Z_k$ !=$A_m$，蕴含“$Z_0$，$Z_1$，…，$Z_{k}$”是“$A_0$，$A_1$，…，$A_{m - 1}$”和“$B_0$， $B_1$，…， $B_{n}$”的一个LCS

  • 3.如果$A_m$ != $B_{n}$，则$Z_k$ != $B_{n}$，蕴含“$Z_0$，$Z_1$，…，$Z_{k}$”是“$A_0$，$A_1$，…，$A_{m}$”和“$B_0$， $B_1$，…， $B_{n - 1}$”的一个LCS

令$f{_i,_j}$表示$A_i$ 及以前和$B_j$及以前的LCS长度，有 $f{_i,_j} = f{_i - _ 1,_j - _ 1} + 1(A{_i}=B{_j})$ $f{_i,_j} = \\max f{_i - _ 1,_j},f{_i,_j - _1}(A{_i}!=B{_j})$ Q.E.D.

代码

```c++ #include #include #include using namespace std; const int N = 100005; char a[N],b[N]; int dp[N][N]; int main() { int lena,lenb,i,j; scanf(“%s%s”,a,b); memset(dp,0,sizeof(dp)); lena=strlen(a); lenb=strlen(b); for(i = 1;i <= lena;i++) { for(j = 1;j <= lenb;j++) { if(a[i - 1] == b[j - 1]) { dp[i][j] = dp[i - 1][j - 1] + 1;// 续上了 LCS加长 } else { dp[i][j] = max(dp[i - 1][j],dp[i][j - 1]);// 没续上 状态转移 } } } printf(“%d\\n”,dp[lena][lenb]); return 0; } 

## T6 合唱队形 ### 题干 照例挂OJ链接 [洛谷P091](https://www.luogu.com.cn/problem/P1091) N 个元素，去除N-K个，使得剩下的 K 个元素中间向两边递减。 已知所有 N 个元素，计算最少需要去除几个，可以使得剩下的元素满足条件。 **输入** 第一行，一个数n。 第二行，n个元素。 **输出** 一行，去除个数。 ### 思路 两边扫LIS，找合理的中间位置。 ### 代码 不解释 ```c++ #include #include using namespace std; int a[10005],len1[10005],len2[10005]; int main() { int n; scanf(“%d”,&n); for(int i = 1; i <= n; ++i) scanf(“%d”,&a[i]); int ans = 0; for(int i = 1;i <= n;i++) {// 一侧 int maxlen = 0; for(int j = 1; j < i;j++) { if(a[j] < a[i]) maxlen = max(maxlen, len1[j]); } len1[i] = maxlen + 1; } for(int i = n;i >= 1;i—) {// 另一侧 int maxlen = 0; for(int j = n; j > i;j—) { if(a[j]<a[i]) maxlen=max(maxlen,len2[j]); } len2[i] = maxlen + 1; } for(int i = 1;i <= n;i++) ans = max(ans,len1[i] + len2[i] - 1); printf(“%d”,n-ans); return 0; }

如题 ```c++ #include #include using namespace std; const int INF = 2147483647,MAXN = 10000; int num[MAXN + 5],b[MAXN + 5],dp[MAXN + 5]; int main() { int n; scanf(“%d”,&n); for(int i = 0;i < n;i++) scanf(“%d”,&num[i]); b[0] = 0; for(int i = 1 ;i <= n ;i++) b[i] = INF; for(int i = 0 ;i < n ;i++) { int pos = upper_bound(b,b + n + 1,num[i]) - b; b[pos] = min(b[pos],num[i]); dp[i] = pos; } int ans = 0; for(int i = 0;i < n;i++) ans = max(ans,dp[i]); printf(“%d”,ans); return 0; } 
## T7 严格递增序列 ### 题干 数列 $A_0$，$A_1$，…，$A_{n}$，去掉最少的数字，使得数列严格单调递增。 **输入** 第一行，一个整数n。 第二行，n个整数。 **输出** 一行，最少删除数。 ### 思路 版子题。 ### 代码 ```c++ #include #include using namespace std; const int INF = 2147483647,MAXN = 10000; int num[MAXN + 5],b[MAXN + 5],dp[MAXN + 5]; int main() { int n; scanf(“%d”,&n); for(int i = 0;i < n;i++) scanf(“%d”,&num[i]); b[0] = 0; for(int i = 1 ;i <= n ;i++) b[i] = INF; for(int i = 0 ;i < n ;i++) { int pos = upper_bound(b,b + n + 1,num[i]) - b; b[pos] = min(b[pos],num[i]); dp[i] = pos; } int ans = 0; for(int i = 0;i < n;i++) ans = max(ans,dp[i]); printf(“%d”,n - ans); return 0; }
